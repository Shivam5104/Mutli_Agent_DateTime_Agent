# Multi Agent DateTime Agent

## Project Description
This project implements a multi-agent system that efficiently interacts with date and time functionalities. It utilizes a combination of various algorithms and techniques to offer users the ability to manage and manipulate date and time data seamlessly.

## Setting Up Google API Key
To run this project locally, you need to set up a Google API key:

1. Go to the [Google Cloud Console](https://console.cloud.google.com/).
2. Create a new project by clicking on 'Select a project' and then 'New Project'.
3. Navigate to the 'API & Services' section.
4. Click on 'Credentials' in the left-hand menu.
5. Click on 'Create credentials' and select 'API key'.
6. Copy the generated API key.
7. Navigate to your local project directory and create a `.env` file.
8. Add your API key in the format: `GOOGLE_API_KEY=your_api_key_here`.
9. Make sure to install necessary dependencies and run the project.

## Running the Project
After setting up the API key, you can run the project using the following command:

```bash
python main.py
```

Make sure to have all dependencies installed and configured properly before running the project.
